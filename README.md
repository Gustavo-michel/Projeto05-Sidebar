# 📂 Sidebar de Navegação

Este projeto é uma página web que implementa uma **sidebar responsiva** (menu lateral), contendo links de navegação, separadores de seções e um painel de informações do usuário.

## 🚀 Tecnologias utilizadas

- **HTML5** → estrutura do projeto  
- **CSS3** → estilização do layout no arquivo `style.css`  
- **Material Symbols (Google)** → ícones do menu  
- **Imagens personalizadas** → ícone da aba e foto de perfil  

## 📋 Funcionalidades

- Sidebar com **links organizados em seções**:
  - *Main Menu* (Dashboard, Overview, Analytic)  
  - *General* (Projects, Groups, Transfer, All Reports, Notifications)  
  - *Account* (Profile, Settings, Logout)  
- **Separadores visuais** para organizar o menu.  
- Exibição de **perfil do usuário** com foto, nome e cargo.  
- Preparado para receber **ícones do Material Symbols** para enriquecer a UI.  

## 🔗 Demonstração

Abra o arquivo `index.html` no navegador ou publique em serviços como:  
- [GitHub Pages](https://pages.github.com/)  
- [Vercel](https://vercel.com/)  
- [Netlify](https://www.netlify.com/)
